连接步骤：

              
    
                
  1. 开启开发者选项：设置-关于手机-连续点击版本号
    2. 
                
  2. 打开USB调试：设置-系统和更新-开发者选项-USB调试
    3. 
                
  3. 数据线将手机连接电脑（必须是支持数据传输的线）
    4. 
                
  4. 拉下通知栏，将"仅充电"改为"传输文件"
    5. 
                
  5. 弹出"是否允许USB调试"点击确定
    6. 
                
  6. 不弹授权大概是驱动问题，安装华为（荣耀）手机助手连接
    7.

  • 
                  error:device not found -
                  USB调试未连接成功，请检查连接
                
    • 
                
  • 
                  Failure [not installed for 0] -
                  手机未安装此包名的APP，请忽略
                
    • 
                
  • 
                  手机ROM中本就没有的，无法通过工具装回，只能装回原本就有的app
                
    • 
                
  • 使用完毕后建议重启一次手机
